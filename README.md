# Way to Billion

Way to Billion ist eine lokale Web-App für kleine Projektteams.

## Ziel

Die App soll:

- Repositories übersichtlich darstellen
- Markdown-Dokumentation mitverfolgen
- Fortschritt und Änderungen für spätere Doku festhalten
- Projektzusammenfassungen vorbereiten

## Geplantes MVP

- Repo hinzufügen
- Repo scannen
- relevante Markdown-Dateien erkennen
- letzte Commits anzeigen
- Activity Log aufbauen
- Summary für Doku/Abgabe erzeugen

## Tech-Stack

- Next.js
- TypeScript
- Tailwind CSS
- später: Prisma + SQLite

## Start

```bash
npm install
npm run dev
```

Danach im Browser `http://localhost:3000` öffnen.
